# ExoplanetTransitPredictions
ExoplanetTransitPredictions — это специализированный инструмент, предназначенный для прогнозирования времени и параметров транзитов экзопланет. Он помогает астрономам планировать наблюдения, вычисляя моменты прохождения планеты перед диском звезды на основе данных каталогов экзопланет.

## Описание

Все предустановленные параметры в файле main.py. Этот файл из запускает расчет.

Обычно требуется только заполнить 
start_date и end_date, можно сразу на весь текущий год.

формат файла с параметрами экзопланет obj.dat

Можно скачать с сайта www.openexoplanetcatalogue.com все xml-файлы и вычислять параметры для интересуемых планет, используя параметры непосредственно из xml. Тогда нужно сделать загрузку требуемых xml вручную или добавтить все через цикл

 listObjectOfInterest.parseXmlFile( "WASP-107.xml" )
 listObjectOfInterest.parseXmlFile( "WASP-48.xml" )

Новую обсерваторию можно легко добавить в observatory.py
